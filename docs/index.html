Project Zomboid

This Is How You Died!

Project Zomboid is a hardcore open-world survival simulator developed by The Indie Stone. Set in the fictional Knox County, Kentucky, the game throws players into a brutal zombie apocalypse with one simple objective — survive for as long as possible.

There are no heroic cutscenes, no last-second rescues, and no guaranteed happy endings. A single bite means infection. Poor planning means starvation. Carelessness means being surrounded. And sometimes, your own overconfidence is more dangerous than the undead.

The game features deep survival mechanics including base building, farming, crafting, medical treatment, mood management, and a detailed skill progression system. Every decision has consequences, and death is permanent.

In Project Zomboid, you don’t win.

You just try to delay the inevitable.
